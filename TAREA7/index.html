Formas de agregar estilos a nuestra Pagina Web

Estilos en Linea

Estos estilos se agregan directamente en el elemento html, mediante el atributo style

Este parrafo tiene estilos en linea.

Estilos internos

Estos estilos se agregan en el mismo archivos html, dentro de la etiqueta <head> y se utiliza la etiqueta <style>

Enlaces Externos

Estos estilos son los que se recomiendan, ya que ofrecen modularidad, facil mantenimiento, orden, eficiencia,entre otras ventajas, para agregar una hoja de estilos externa, dentro de la etiqueta <head>, se va a utilizar la etiqueta <link>
