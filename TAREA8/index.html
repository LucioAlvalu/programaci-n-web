Colores multinavegadores

Hay mas de 16o colores nombrados que son compatibles con todos los navegadores modernos. Estos nombres se pueden usar directamente en css para aplicar colores a elementos sin especificar valores hexadecimales, rgb o hsl.

Ejemplo

  1. Negro
  2. Blanco
  3. Salmon
  4. Azul
  5. Verde

Colores de un borde

Para que a un elemento html se le pueda aplicar color a su borde son necesarias 2 propiedades: border-style y border-color
