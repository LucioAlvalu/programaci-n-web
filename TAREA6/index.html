Lenguaje de marcado de hipertexto (HTML)

Bienvenidos, Hoy estaremos practicando algunos conceptos y elementos mas importantes Vistos hasta el momento en la materia de Programacion Web 1, ademas veremos el uso de etiquetas semanticas en la estructuracion de un sitio web.

conceptos principales de HTML

HTMl

lenguaje de marcado de hipertexto que se utiliza para hacer una pagina web, es el esqueleto.

elemento HTML

Es la unica basica de una estructura y contenido de una pagina Web que esta compuesta de:

  1. Etiqueta de apertura
  2. Pares Atributo-Valor
  3. Contenido
  4. Etiqueta de cierre

Etiqueta <head>

Es una etiqueta contiene metadatos, es la forma en que una pagina Web se comunica con el navegador, funciona para el SEO, se puede enlazar archivo CSS o Script.

Etiqueta <body>

contiene todo el contenido HTML

contenedores semanticos

<header> Etiqueta utlizada para el contenido introductorio de una pagina web puede contener el logo el menu de navegacion principal, el nombre de la empresa etc.
<main> Etiqueta unica de una pagina web cuyo proposito es resaltar nuestro objetivo o mensaje principal del sitio web, dentro se colocara el contenido principal de nuestra pagina
<section> Es una etiqueta que poco a poco se intenta que sutituya a <div> su uso es para dividir la pagina en secciones y agrupar contenido relacionado
<article> Es una etiqueta que se utiliza para agrupar contenido autonomo que si se separa de otro contenido sigue teniendo coherencia normalmente se compone de un encabezado y contenido
<footer> Es una etiqueta que normalmente se encuentra hasta el final de una pagina y funciona como cierre del sitio web, a menudo contiene datos de contacto, derechos de autor u otra informacion complementada

Etiquetas principales HTML

Etiqueta<a>

Es un elemento que nos va a permitir navegar a:

  • Paginas Externas
  • Paginas Internas
  • Secciones dentro de nuestra misma pagina (enlaces ancla)

Alguno valores especificos para el atributo href son:

href="https://wa.me/5613738095"
Abre un chat del numero especificado en Whatsapp
href="mailto:example@gmail.com"
Abre una aplicacion de correo, con el destinatario especificado
